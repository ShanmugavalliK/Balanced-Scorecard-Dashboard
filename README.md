# Balanced Scorecard Dashboard

## Project Overview
The Balanced Scorecard Dashboard is an interactive Power BI project developed to monitor organizational performance using strategic KPI tracking and weighted score analysis. The dashboard evaluates business performance across four major perspectives of the Balanced Scorecard framework:

- Financial
- Customer
- Internal Processes
- Learning & Growth

The project helps organizations measure strategic objectives, monitor KPI performance, and identify areas requiring improvement through dynamic visualizations and score-based performance tracking.

---

## Objectives
- Monitor overall organizational performance
- Track KPI status across strategic business areas
- Compare actual performance against targets
- Analyze weighted score contributions
- Support data-driven strategic decision-making
- Improve executive-level KPI visibility

## KPI Analysis Pages
Each KPI includes:
- Strategic measure description
- KPI target details
- Actual vs Target comparison
- Variance analysis
- Weighted KPI score
- Year-to-date tracking visuals

---

## Score Calculation
The weighted score calculation is based on:

```text
((Worst Case - Actual Value) / (Worst Case - Best Case)) * Weighting
```

This formula helps evaluate KPI performance contribution toward the overall organizational score.
